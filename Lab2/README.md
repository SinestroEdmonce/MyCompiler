# 程序功能
- [x] 实验代码已实现对C语言代码的简单词法分析及语法分析，能够识别基本的词法错误和语法错误，例如：a[..]属于对数组声明的错误等。

- [x] 代码中符号表采用了基于十字链表和Open hashing的散列表结构。
